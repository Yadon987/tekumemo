# 卒業制作提出概要：てくメモ ～てくてくメモリア～

## Webアプリの概要

「てくメモ ～てくてくメモリア～」は、デスクワークで運動不足になりがちなエンジニアやビジネスパーソンに向けた**散歩習慣化コミュニティプラットフォーム**です。

### 【課題】
長時間の座り作業による運動不足は、体力低下だけでなくメンタル不調や生産性低下を招きます。しかし、目的のない単調な散歩を一人で継続するのは困難です。

### 【解決策】
Google Fit連携により、スマホを持って歩くだけで歩数・距離・カロリーを完全自動記録。面倒なデータ入力は一切不要です。

### 【使用イメージ】
仕事終わり、30分の散歩に出かけると、帰宅後にGoogle Fitから自動でデータが同期。「今日は3.2km歩いた！」と記録され、目標達成率がグラフで表示されます。散歩記録がカレンダーに記録され、継続日数が可視化されます。投稿すると、仲間から「お疲れ様！」のスタンプリアクションが届き、ランキングでは同僚と歩数を競い合えます。条件クリアで称号バッジを獲得し、SNSにOGP画像付きでシェアすることも可能です。

### 【特徴】
「面倒なことは全て自動化」を徹底。習慣化の鍵となる「継続スタンプカレンダー」と、仲間の頑張りが届く「SNS機能」により、孤独な運動をチーム戦のような一体感へ昇華。エンジニアらしい効率的な健康管理と、温かみのあるコミュニティ体験を両立しました。
