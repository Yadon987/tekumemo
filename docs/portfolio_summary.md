# 🚶‍♂️ てくメモ (Tekumemo) - 散歩記録 & SNS アプリケーション

## 💡 アプリケーション概要

「毎日の散歩を、もっと楽しく、もっと特別に。」
日々の散歩記録を管理するだけでなく、その時の「気分」や「天気」をシェアし、ユーザー同士が緩やかにつながる SNS 機能を搭載した Web アプリケーションです。
単なる記録ツールではなく、**「使うこと自体が楽しい」** と感じられる UI/UX を目指し、徹底的なデザインの作り込みを行いました。

## 🛠 技術スタック

- **Backend:** Ruby 3.2 / Ruby on Rails 7.0
- **Frontend:** Hotwire (Turbo/Stimulus), Tailwind CSS
- **Database:** PostgreSQL
- **Testing:** RSpec (Model / System / Request Specs), FactoryBot
- **Linting:** RuboCop
- **Environment:** Linux (Development)

## ✨ こだわりのポイント（技術・デザイン）

### 1. モダンなフロントエンド実装と高度な CSS 設計

Rails 7 の標準である**Hotwire**をフル活用し、SPA のような高速な画面遷移と部分更新を実現しました。
特にデザインには力を入れ、**Tailwind CSS**を駆使して 2 つの異なる世界観を実装しました。

- **Light Mode (Puni-Puni Claymorphism):**
  - 粘土のような柔らかい質感、複数の影(`box-shadow`)を組み合わせた立体感、パステルカラーによる親しみやすさを表現。
- **Dark Mode (Neon Noir / Synthwave):**
  - 夜のネオン街をイメージした発光表現、ガラスのような透明感(`backdrop-blur`)、深い紫と鮮やかなアクセントカラーによる没入感を演出。
- **実装の工夫:**
  - `dark:` プレフィックスを活用し、HTML 構造を変えずにクラスの切り替えだけで全く異なるテーマを実現。
  - 背景アニメーション（浮遊するオーブ）を CSS Animation で実装し、リッチな体験を提供。

### 2. ユーザー体験(UX)を重視した機能設計

- **直感的な投稿フロー:**
  - テキストだけでなく、「天気」や「気分」をアイコンで直感的に選択できる UI。
  - モーダルウィンドウを採用し、ページ遷移なしでスムーズに投稿可能。
- **緩やかなつながり（リアクション機能）:**
  - 言語化しにくい感情もスタンプ（絵文字）で気軽に伝えられるリアクション機能を実装。
  - ポップオーバー UI を Stimulus で自作し、インタラクティブな操作感を実現。

### 3. 品質保証とテスト駆動開発

- **RSpec による網羅的なテスト:**
  - **Model Spec:** バリデーションやアソシエーション、複雑なロジック（一意性制約など）の検証。
  - **System Spec (E2E):** ユーザー操作をシミュレートし、JavaScript を含む動的な挙動（モーダル開閉、非同期通信）を保証。
- **堅牢な設計:**
  - RuboCop による静的解析を導入し、コードスタイルの統一と品質維持を徹底。
  - N+1 問題の回避や、適切なインデックス設計によるパフォーマンスへの配慮。

## 🚀 今後の展望

- **Google Fit 連携:** 歩数データの自動取得（MVP 後のフェーズで実装予定）。
- **PWA 化:** モバイルでのネイティブアプリに近い体験の提供。
- **通知機能:** リアクションやコメントのリアルタイム通知。
