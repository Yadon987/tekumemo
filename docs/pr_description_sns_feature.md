## 概要

「みんなの足跡」として、ユーザー間で散歩の記録や気分を共有できる SNS 機能を実装しました。
投稿機能に加え、Slack 風のリアクション機能、削除機能、そして没入感を高める動的な UI デザインを取り入れています。

## 変更点

### ✨ 新機能・UI/UX 改善

- **投稿機能**: テキスト、気分、天気、散歩記録を紐付けて投稿可能に。
- **リアクション機能**:
  - Slack 風のモダンなリアクションピッカーを実装。
  - ボタンを左端に固定し、操作性を向上。
  - Turbo Stream を使用したリアルタイム更新（Optimistic UI）。
- **削除機能**:
  - 投稿者本人のみに表示される削除ボタン（ゴミ箱アイコン）を右上に配置。
  - モバイルでも押しやすいよう常時表示に対応。
- **動的背景デザイン**:
  - 「みんなの足跡」ページ専用の動的背景（光のオーブが明滅するアニメーション）を実装。
  - ヘッダーの透明度を調整し、グラスモーフィズム効果を強化。
  - ページ遷移時のチラつきを防止するため、背景色の設計を見直し。

### 🛠 技術的修正・品質向上

- **Lint/Security**:
  - RuboCop によるコードフォーマットの修正（全指摘解消）。
  - Brakeman によるセキュリティチェック（動的レンダリングの警告対応）。
- **Refactoring**:
  - Rails 8.0 を見据え、`enum` 定義を推奨形式（位置引数）に修正。
  - リアクション表示のループ処理バグを修正。
- **Test**:
  - RSpec（System Spec / Model Spec）の追加とパス確認。
