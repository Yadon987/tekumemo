## 📝 概要

ランキング OGP 画像の生成ロジックのリファクタリングと、画像ファイルの最適化を行いました。
コードの重複を排除し、画像の読み込み速度を向上させることを目的としています。

## ✨ 変更内容

### ♻️ リファクタリング

- **User モデルへのロジック移動:**
  - ランキング集計ロジック（順位計算、歩数・距離集計）を `User#weekly_ranking_stats` メソッドとして切り出しました。
- **重複コードの削除:**
  - `Rankings::OgpImagesController` と `GenerateRankingOgpImageJob` で重複していた集計処理を削除し、モデルのメソッドを呼び出すように変更しました。

### 🚀 パフォーマンス最適化

- **画像圧縮率の調整:**
  - `MiniMagick` の `quality` 設定を `80` から `75` に変更しました。
- **メタデータ削除:**
  - `strip` メソッドを追加し、画像に含まれる不要なメタデータ（Exif 情報など）を削除してファイルサイズを軽量化しました。

## ✅ 確認事項

- [x] ランキング OGP 画像が正常に生成されること
- [x] 生成された画像の画質が許容範囲内であること
- [x] テストが全てパスすること (`rspec spec/jobs/generate_ranking_ogp_image_job_spec.rb spec/requests/rankings/ogp_images_spec.rb`)
